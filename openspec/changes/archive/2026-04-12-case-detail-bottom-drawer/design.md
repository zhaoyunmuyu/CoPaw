## Context

聊天首页已有精选案例卡片（FeaturedCases），悬浮时显示「看案例」和「做同款」按钮。目前「看案例」按钮无实际功能。需要实现点击后从底部弹出案例详情弹窗，按 Pixso 设计稿还原 UI。

设计稿显示弹窗内容为金融案例（他行存款到期潜力客户名单），包含左右分栏：左侧数据表格、右侧步骤说明，底部有操作按钮。

## Goals / Non-Goals

**Goals:**
- 从页面底部滑入的案例详情弹窗，覆盖页面主体区域
- 弹窗顶部标题栏（48px）：左侧"案例详情"文字，右侧关闭按钮
- 左右分栏内容区：左侧展示案例数据表格，右侧展示步骤说明
- 底部操作栏（60px）：「订阅为定时任务」和「做同款」两个胶囊按钮
- 「做同款」点击后关闭弹窗，将案例描述填入聊天输入框
- 弹窗有半透明遮罩，点击遮罩或关闭按钮可关闭

**Non-Goals:**
- 不做数据动态加载（案例内容先硬编码或从 FeaturedCases props 传入）
- 不做「订阅为定时任务」的实际功能（仅 UI 占位）
- 不修改顶部导航栏

## Decisions

1. **使用 Ant Design Drawer 组件**：从底部弹出，利用现有组件能力，placement="bottom"，自定义高度占页面 ~90%
2. **弹窗内容通过 props 传入**：FeaturedCases 卡片数据中包含 detailContent，点击「看案例」时传给弹窗
3. **左右分栏布局**：使用 flex 布局，左侧 60%（数据表格），右侧 40%（步骤说明），中间有间距
4. **状态管理**：在 Chat/index.tsx 中管理弹窗开关状态和当前选中的案例数据，通过 props 传递给 FeaturedCases 和 CaseDetailDrawer

## Risks / Trade-offs

- 案例内容目前硬编码在组件内，后续需对接后端 API 获取真实案例数据
- 弹窗高度固定为页面高度的大部分，在小屏幕上可能需要滚动
